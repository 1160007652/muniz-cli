## @muniz/cli

Muniz 脚手架, 是一个基于插件机制模型设计的, 由多个插件组合而成。

### 项目框架

#### Lerna

- 采用 workspace 模型
- 结合 yarn - workspace 管理 node_modules

#### Eslint

- 使用 Eslint 进行代码规范校验
- 继承 于 airbnb 规范

#### Prettierrc

- 使用 Prettierrc 进行代码本地、提交时的格式校验
- 与 Eslint 冲突的解决

#### Commitlint

- 进行 Github 代码提交 Commit 校验
- husky 、 lint-staged

#### ink

- React 开发命令行库

#### meow

- 命令行解释器
